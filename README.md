# Smart Energy Consumption Optimizer

## Project Overview
This project aims to build an intelligent system that predicts, analyzes, and optimizes household electricity consumption using machine learning techniques.

## Objectives
- Forecast energy consumption
- Detect anomalies
- Analyze usage patterns
- Recommend cost optimization strategies

## Models Used
- Linear Regression
- Random Forest Regressor
- ARIMA
- K-Means Clustering
- Isolation Forest

## Project Structure
- data/ → Dataset files
- notebooks/ → Jupyter notebooks
- models/ → Saved trained models
- reports/ → Documentation
- presentation/ → Project slides
